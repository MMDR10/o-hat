# Ô Classification Spectrum v2.0

## The Seven System Types

| Balance Ratio | System Type | Characteristics | Verified Examples |
|:------------:|:------------|:----------------|:------------------|
| **> 20×** | **SUPER-PULSE** | Extreme contrast: near-perfect baseline + moderate outbreak | 1989 Quebec geomagnetic storm (40.3×) |
| **5–15×** | **PULSE** | Sparse baseline (mostly zero events) | Earthquakes M5+ (9.2×) |
| **1.5–4×** | **CONTINUOUS FLOW** | Baseline has regular activity | ENSO (2.2×), COVID-19 (2.5×), Typhoon storms (2.6×) |
| **< 1.5×** | **REGIME SATURATION** | System too periodic/forced | Typhoon seasons (0.4×) |
| **Mixed** | **VOLATILITY SHIFT** | Balance < 1, helicity up | S&P 500 crashes |
| **Mixed** | **COLLAPSE (Type X)** | H_v3 down (structure destruction) | Geomagnetic storms, microservice failures |
| **Mixed** | **PERTURBATION** | H_v3 up (structure emergence) | ICS attacks, LLM jailbreaks |

## Scale-Dependent Transitions (v2.0)

Some systems exhibit **different types at different scales**:

| System | Micro Scale | Macro Scale | Mechanism |
|--------|:----------:|:----------:|:----------|
| **Typhoons** | Continuous Flow (2.6×) | Regime Saturation (0.4×) | External seasonal forcing |
