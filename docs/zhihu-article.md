# 拒绝万亿参数的沉重防御：如何用 3 个数学指标，纯靠几何结构揪出 AI 越狱与网络攻击？

## 一、你花几块钱攻击，我花几十万防御——这仗没法打

现在的 AI 安全，打的是一个荒唐的算力不对称战争。

攻击者用一张消费级显卡、几块钱的电费，拿开源模型生成越狱 prompt，几秒钟完事。防御方呢？要部署同等甚至更强的模型，7×24 实时扫描所有输入输出。GPT-4 级别的实时过滤，一年下来几十万人民币的推理成本。

更离谱的是——攻击者把 prompt 用 Base64 编个码、换几个同义词、加一段"奶奶哄我睡觉"的角色扮演，你的大模型防御就哑火了。

| 方案 | 成本 | 弱点 |
|:--|:--|:--|
| 大模型实时过滤 | $30K-$5M/年 | 成本是攻击者的 100 倍 |
| 关键词黑名单 | 几乎免费 | Base64 变形直接绕过 |
| BERT 轻量分类器 | 中等 | 对抗样本脆弱，需持续重训练 |

这不是技术问题，是经济问题。你在一个**攻击者永远比你便宜**的赛道上跟人拼消耗。

## 二、所以，我们不玩了。换赛道。

我们做了一件事：**不从语义层面检测攻击，从几何层面检测攻击。**

什么意思？

攻击者可以改写 prompt 的内容。但他改不了攻击行为在相空间（phase space）中留下的几何轨迹。Base64 编码改变了字面形式，但 jailbreak 的结构性特征——系统轨迹突然变得螺旋缠绕、平衡崩塌——在几何读数上完全一致。

我们定义了三个纯数学指标：

| 指标 | 量度内容 | 为什么无法绕过 |
|:--|:--|:--|
| **curl（旋度）** | 系统轨迹的方向变化率 | 攻击必伴随急转弯，内容改不了轨迹 |
| **helicity（螺旋性）** | 系统内部的结构复杂度 | 越狱导致语义空间剧烈缠绕 |
| **balance（平衡）** | 系统距离临界点的距离 | 攻击是系统崩塌，崩塌有几何指纹 |

**核心洞察：Õ 量度的是攻击的「物理结构」，不是攻击的「语义内容」。**

这就是护城河。

## 三、把万亿参数的问题，降维成几行 CPU 代码

最狠的地方在这：**Õ 不需要 GPU。不需要训练。不需要理解 prompt 说了什么。**

它就是三行数学。单核 CPU。几毫秒。可以在 NAS、IoT 网关、甚至手机 NPU 上跑。

| | 传统防御 | Õ |
|:--|:--|:--|
| 检测对象 | 语义内容（说了什么） | 几何结构（怎么说的） |
| 算力需求 | GPU 集群 | CPU 单核，几毫秒 |
| 部署场景 | 云端 | 边缘设备、NAS、手机 |
| 绕过难度 | 低（改写法） | 高（结构涌现无法伪装） |

一个需要万亿参数模型才能理解的防御问题，被降维成一个在入门级 NAS 上靠几行 Python 就能实时运行的几何测量。

```bash
pip install numpy scipy
python o_hat.py your_data.csv
```

## 四、一个框架，跨十二个领域——同一把尺，同一个结构

Õ 最让我们自己都意外的地方是：它不是 AI 安全专用的。同一套方法、同一个脚本、不调参数，甩到完全不同的领域——全部跑通了。

| 领域 | 事件 | Balance Ratio | 系统类型 |
|:--|:--|:--|:--|
| 🌊 气候 | ENSO 厄尔尼诺（1950-2025） | 2.33× | CONTINUOUS FLOW |
| 🌍 地震 | 台湾 M5+ 地震 | 7.33× | PULSE |
| ⚡ 地磁暴 | 1989 Quebec 大停电 | **40.3×** | SUPER-PULSE |
| 🦠 疫情 | COVID-19 爆发 | 2.6× | CONTINUOUS FLOW |
| 🧠 AI 安全 | LLM 越狱检测 | 5-11× | PULSE |
| 📈 金融 | 台湾加权指数 | 1.8× | CONTINUOUS FLOW |

**三个数字。十二个领域。一把尺。**

数学不关心你测的是海水温度、病例数量、还是股价——梯度和偏度在任何有序序列中都存在。Õ 就是利用了这一点：把不同领域的测量统一到一个几何语言里。

### 而且这不是 cherry-picking

我们做了 **11×11 的 grid sweep**：窗口大小和基线长度各从 50% 扫到 150%，总共 121 种参数组合。所有组合下，Balance Ratio 的均值漂移不到 15%。这个结构信号不是调参调出来的——它是客观存在的。

## 五、AI 越狱实测：5 倍螺旋飙升，纯几何判读

最硬核的证据在 LLM 越狱检测上。

| 指标 | 正常 Prompt | 越狱 Prompt | 效果 |
|:--|:--|:--|:--|
| Storm Zone H_v3（螺旋性） | 3.81 | **18.20** | ↑ **5×** |
| Peak-to-Storm Ratio | 37.5 | 3.34 | ↑ **11×** |
| L27/Sig Ratio | — | **-2.31** | 最强鉴别器 |

当 LLM 遭遇越狱攻击时，它的 hidden states 轨迹不是渐渐偏离——是**突然螺旋化**。这不是语义判断，是你可以画在纸上的几何事实。

而且这个信号在 Qwen2.5 系列（0.5B/1.5B/3B/7B/14B）上全部复现。模型大小不影响结构涌现——越狱本身的「物理特征」是一致的。

## 七、为什么这件事重要

2026 年的 AI 安全，正在重蹈网络安全早期的覆辙：**所有人都想在同一个维度上跟攻击者比谁力气大。**

但真正的突破口从来不在「更大的盾」——在**换维度**。

密码学不是靠更长的密钥赢的，是靠数学上单向函数的不可逆性。同理，Õ 不是靠更大的模型赢的，是靠**几何结构在语义伪装下的不变性**。

攻击者可以改写 prompt。可以 Base64。可以奶奶哄睡。但他在相空间中留下的轨迹——那个急转弯、那个螺旋爆发、那个平衡崩塌——删不掉。

**你可以在语义层面撒谎。你在几何层面说不了谎。**
